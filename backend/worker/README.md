# Worker-Pool
To run this application you need at least `Go 1.25.5`

## How to run
On the project root, paste de following command:
```bash
go mod tidy
go run .
```

If you want to run the binary execution, just do:
```bash
go build -o bin/main.<extension-name-of-your-OS>
bin/main.<extension-name-of-your-OS>
```
